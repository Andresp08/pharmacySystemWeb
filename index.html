Mantenimiento entidad Categorias

Para añadir una nueva categoria se debe presionar el botón nuevo, para que se nos habiliten los campos, hecho esto podremos digitar los datos y pinchar el botón guardar, para añadir el registro a la base de datos.
Si lo que queremos es eliminar un registro, debemos seleccionar de la tabla el dato que queremos eliminar, y posteriormente presionar el botón para ejecutar la acción.

Ventas

Proceso de venta de productos.

Para realizar el proceso de venta, debemos escoger un cliente hacía el cual va dirigida, posteriormente pinchamos el botón Buscar Producto y escogemos el deseado.

1) despues de realizar estre proceso pinchar el boton agregar en la parte derecha para añadir el producto al proceso de venta, si se desea escoger otro producto, nuevamente se debe repetir este proceso.

2) si lo que se desea es eliminar algun producto, se selecciona el respectivo desde la tabla de listado y se le pincha en el botón eliminar.

3) Una vez realizado todo el proceso, se debe tocar el botón generar venta, recibir el dinero y entregar el respectivo ticket

  • Entidad: Ventas, Detalle Ventas
  • Ilustracion: Ventas
